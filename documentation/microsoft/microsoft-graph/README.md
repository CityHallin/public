# Microsoft Graph
To gather data in Microsoft 365, Microsoft is moving to its newer technology for this called [Microsoft Graph](https://learn.microsoft.com/en-us/graph/overview). 

To get started, review the doc here for [Install the Microsoft Graph PowerShell SDK](https://learn.microsoft.com/en-us/powershell/microsoftgraph/installation?view=graph-powershell-1.0).
